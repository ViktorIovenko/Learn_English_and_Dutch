# Важные задачи

## 1. Критический баг: приложение вылетает при нажатии «Настройки»

Приоритет: исправить в первую очередь.

### Проблема

При нажатии на кнопку или ссылку `Настройки` приложение вылетает.

### Что проверить

- обработчик нажатия;
- route / navigation;
- Activity;
- Fragment;
- экран настроек;
- возврат назад.

### Ожидаемое поведение

- пользователь нажимает «Настройки»;
- приложение не вылетает;
- открывается экран настроек;
- возврат назад работает нормально.

# Режим изучения слов

## Верхний переключатель языков: добавить режим ALL

Сейчас сверху есть переключатель:

NL / EN / RU

Нужно добавить режим:

ALL

### Ожидаемое поведение

При выборе ALL показываются одновременно:

- NL
- EN
- RU

Важно:

Такой режим уже реализован в Telegram App.

Нужно использовать существующую логику.

Это не кнопка внутри карточки слова.

Это отдельный режим верхнего переключателя.

## После правильного ответа кнопка должна стать «Следующее слово»

### Сейчас

Пользователь:

- вводит слово;
- нажимает «Проверить»;
- экран становится зелёным;
- для перехода нажимает маленькую стрелку.

### Нужно

После правильного ответа:

Кнопка:

Проверить

должна превращаться в:

Следующее слово

или:

Дальше

### Ожидаемое поведение

Пользователь:

Проверил → нажал эту же кнопку → следующее слово.

Не нужно тянуться к стрелке.

## Неправильный ответ без всплывающего окна

### Сейчас

При ошибке показывается popup.

В popup:

- слова;
- предложения.

Это ломает поток обучения.

### Нужно

Убрать popup.

После ошибки:

- буквы автоматически перемещаются на правильные места;
- пользователь сразу видит ошибку;
- появляется предложение на изучаемом языке;
- кнопка становится:

Следующее слово

# Главная страница

## Показывать процент известных слов

На карточке урока показывать:

- последнее открытие;
- процент:

50%
70%
85%

- небольшой прогресс.

---

## Логика подсчёта знаний

Важно:

Пропуск ≠ знает слово.

### Правила

Слово становится известным только после первого правильного ввода.

### Правила:

Пропуск до первого правильного ответа:

ничего не даёт.

Первый правильный ответ:

слово становится известным.

Если после этого пользователь пропустил:

статус сохраняется.

Если позже пользователь ввёл неправильно:

статус сбрасывается.

---

### Пример

10 слов.

Пользователь правильно ввёл:

7.

Показываем:

70%

Если пользователь потом пропустил эти слова:

процент не уменьшается.

Если позже ошибся:

процент уменьшается.

---

## Визуально показывать какие уроки плохо знает

Важно:

НЕ делать страницу радугой.

Не делать яркие разноцветные карточки.

### Нужен маленький индикатор:

варианты:

- тонкая полоска;
- маленькая точка;
- маленький прогресс;
- аккуратная шкала.

### Цель

С первого взгляда понимать:

какие уроки стоит повторять.
