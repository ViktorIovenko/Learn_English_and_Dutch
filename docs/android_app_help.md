## Правки и задачи

### Режим изучения слов: кнопка показа всех слов на всех языках

**Проблема:**

В Android-приложении в режиме изучения слов сейчас можно скрыть только один язык. Но нет кнопки, которая включает показ всех слов сразу на всех языках.

**Что нужно сделать:**

Добавить кнопку в режим изучения слов, которая будет показывать все варианты слова/идиомы на всех доступных языках.

**Ожидаемое поведение:**

- Пользователь находится в режиме изучения слов.
- По умолчанию часть языков может быть скрыта, как сейчас реализовано.
- При нажатии на новую кнопку приложение показывает все слова/переводы на всех языках.
- Кнопка должна работать аналогично уже реализованной функции в другой версии/части проекта.

**Важно:**

Эта кнопка уже была реализована раньше, но в Android-приложении её сейчас нет. Нужно найти существующую реализацию и перенести/повторить её для Android-приложения.
